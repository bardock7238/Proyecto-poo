# Proyecto-poo
Proyecto de Programacion orientada a Objetos

## Objetivo
El objetivo de estre proyecto es simular un sistema de envios teniendo en cuenta diferentes vehiculos, donde se calcula el costo y tiempo de un envio.

## Requisitos
- Tener instalado Java JDK 17 o superior
- Editor de codigo como Visual Studio Code o Apache Net Beans

## ¿Cómo ejecutarlo?
1. Descargar y descomprimir el zip
2. Abrir la carpeta del proyecto en VS Code
3. Abrir el archivo "src/Main.java"
4. Hacer clic en el boton de run 

## Estructura de archivos
```
proyecto-poo/

│   ├── Bicicleta.java

│   ├── Avion.java
│   └── Main.java

```

## Opciones del menú
1. Calcular costo de envío
2. Calcular tiempo de entrega
3. Validar si la carga es válida para el vehículo
4. Ver reporte comparativo de todos los vehiculos
5. Salir
