# Proyecto-poo
Proyecto de Programación Orientada a Objetos

## Descripción
Este proyecto consiste en el desarrollo de un sistema en Java que simula el cálculo del costo y el tiempo de envío de mercancías. Se aplican principios fundamentales de la programación orientada a objetos, como clases abstractas, herencia y polimorfismo, para modelar diferentes tipos de transporte.

## Objetivo
El objetivo de este proyecto es simular un sistema de envíos teniendo en cuenta diferentes vehículos, donde se calcula el costo y tiempo de un envío.

## Objetivos específicos
- Aplicar los principios de la programación orientada a objetos.
- Implementar clases abstractas y herencia.
- Definir métodos abstractos y su sobrescritura en clases concretas.
- Simular el cálculo de costo y tiempo de envío.
- Comparar diferentes medios de transporte.

## Conceptos aplicados
- Clases abstractas
- Herencia
- Polimorfismo

## Requisitos
- Tener instalado Java JDK 17 o superior
- Editor de código como Visual Studio Code o Apache NetBeans

## ¿Cómo ejecutarlo?

### Opción 1: Descargar el ZIP
1. Descargar y descomprimir el zip
2. Abrir la carpeta del proyecto en VS Code
3. Abrir el archivo `src/Main.java`
4. Hacer clic en el botón de Run

### Opción 2: Clonar el repositorio
1. Clonar el repositorio:
```
   git clone https://github.com/bardock7238/Proyecto-poo.git
```
2. Abrir la carpeta del proyecto en VS Code
3. Abrir el archivo `src/Main.java`
4. Hacer clic en el botón de Run

## Funcionalidades
- Cálculo del costo de envío según distancia y carga.
- Estimación del tiempo de entrega con base en la velocidad.
- Validación de la capacidad máxima de carga.
- Generación de un reporte comparativo entre vehículos.

## Opciones del menú
1. Calcular costo de envío
2. Calcular tiempo de entrega
3. Validar si la carga es válida para el vehículo
4. Ver reporte comparativo de todos los vehículos
5. Salir
